# Server locally

Follow me:

- **Step 1 -** [Download and install Node.js](https://nodejs.org/en/download/)

- **Step 2 -** Clone this repo:

```bash
git clone https://github.com/PhyX-Meow/dotfiles.git

cd dotfiles/

git checkout v2
```

- **Step 3 -** Install dependencies and run locally:

```bash
npm i && npm run dev
```
